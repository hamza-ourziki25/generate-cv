# CV Generator (PHP + MySQL)

Application web qui permet à un utilisateur de :
1) remplir un formulaire,
2) choisir un design de CV,
3) générer automatiquement un CV,
4) enregistrer les informations dans une base de données MySQL.

## Fonctionnalités
- Formulaire utilisateur (infos personnelles, formation, expérience, compétences, etc.).
- Sélection d’un template/design de CV.
- Génération d’un CV (HTML/CSS) prêt à imprimer / télécharger.
- Sauvegarde des données dans MySQL.
- Validation côté front (JavaScript) + traitement côté serveur (PHP).

## Stack
- Frontend : HTML, CSS, JavaScript
- Backend : PHP
- Database : MySQL (SQL)
